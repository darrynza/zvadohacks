# zvadohacks

zvadohacks is tools build with python for web scouting sniffing and pen testing

## Getting Started



### Prerequisites

What things you need to install the software and how to install them

```
- python 2.7
- python 3.0
- git
- pip
```

### Installing


```
clone repo
```
```
cd into cloned repo
```
```
to run a tool use python3 [name of tool] [options]
```
